# Virnix — Output Quality Checklist

Human review checklist for evaluating real AI output quality after each test run.

Score each criterion **1–5** (1 = poor, 3 = acceptable, 5 = excellent).

---

## How to Use

1. Generate output using a real YouTube URL with `NEXT_PUBLIC_FLAG_REAL_AI_GENERATION=true`.
2. Copy each output card into the relevant section below.
3. Score each criterion honestly.
4. If any dimension scores 1–2 repeatedly, note it — it likely points to a prompt weakness.
5. Total score per card: **35 = maximum** (7 criteria × 5).

---

## Scoring Criteria Definitions

| Criterion | What to Look For |
|---|---|
| **Curiosity** | Does the opening create an unanswered question or tension in the reader's mind? |
| **Specificity** | Are there numbers, named frameworks, real scenarios, or concrete details? |
| **Emotional Pull** | Does it connect to fear, ambition, frustration, desire, or surprise? |
| **Platform Tone** | Does it sound native to the platform — not copy-pasted between formats? |
| **Clarity** | Is it immediately clear what this content is about? No jargon fog. |
| **Non-Generic Language** | Would this sentence appear in 1000 other pieces of content? Low = better. |
| **Usefulness** | Would a creator actually use this, or does it need heavy editing? |
