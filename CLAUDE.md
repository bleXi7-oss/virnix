# Virnix

Virnix is a creator-first AI content repurposing SaaS.

Core positioning:
"Turn 1 podcast into 30 viral posts in 60 seconds."

Virnix takes:
- YouTube links
- podcasts
- MP4 videos
- voice memos

Virnix generates:
- TikTok hooks
- short-form scripts
- X/Twitter threads
- LinkedIn posts
- Instagram captions
- YouTube timestamps
- blog summaries
- viral titles

Core philosophy:
- extreme simplicity
- viral-first outputs
- creator-first UX
- minimal
- modern
- fast
- frictionless
- internet-native

Virnix should NOT feel like:
- enterprise software
- corporate SaaS
- complicated dashboards
- feature-bloated tools

Competitive advantage:
- better hooks
- creator psychology
- platform-native outputs
- speed
- simplicity

Tech stack:
- Next.js
- Tailwind
- Supabase
- Vercel
- Stripe

Development philosophy:
- avoid overengineering
- cheapest/simple infrastructure
- MVP first
- fast validation
- fast iteration

Always:
- keep code simple
- keep UI minimal
- prioritize UX clarity
- prioritize speed
- prioritize mobile responsiveness

Important:
Run:
- npm run lint

before finishing changes.
